h1. TERONet

This application was designed to aid Native American TERO Offices in tracking the daily operations of their offices.


h2. Ruby on Rails

This application requires:

* Ruby version 2.0.0
* Rails version 4.0.0

h2. Database

This application uses MySQL with ActiveRecord.

h2. Development

* Template Engine: ERB
* Testing Framework: RSpec and Factory Girl
* Front-end Framework: Zurb Foundation
* Form Builder: SimpleForm
* Authentication: Devise
* Authorization: None
